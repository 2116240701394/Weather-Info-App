Weather Information App
The Weather Information App is a comprehensive meteorological tracking and visualization platform. It provides real-time weather data, severe weather alerts, interactive mapping, and historical record-breaking analysis for users.

Features
User Authentication & Profiles: Secure registration, login tracking, and user-customizable settings (Metric/Imperial units).

Core Weather Data Integration: Fetches real-time data securely via the Azure Maps Weather API with a Cosmos DB caching layer.

Interactive Mapping & Visualization: Displays interactive maps with overlays for radar and satellite data, along with touch and zoom gestures.

Severe Weather Alerts: Real-time push notifications and custom alert thresholds with a historical log of past weather events.

Prerequisites
Ensure you have the following installed on your local machine:

Python 3.10 or higher

Git

Access to an Azure DevOps/Jira workspace for issue tracking
